Homework 12.1

  1. Plot the points `(4,0,3)`, `(3,2,5)`, `(5,-4,-1)`, and `(-3,5,-2)` together in `RR^3` on the same set of axes.
  2. Sketch the following planes in `RR^3` (graph each separately):
    1. `y = 2x + 1`
    2. `y = 2`
    3. `z = 3`
    4. `2x + 3y + z = 6`
  3. Provide a written description of the given region in `RR^3`:
    1. `z > 5`
    2. `0 < z <= 5`
    3. `x^2 + y^2 + z^2 > 4`
  4. Find the distance between the points `(1,-3,-4)` and `(-7,-5,2)`.
  5. Find the equation of a sphere with center `(-5,6,-8)` and radius `7`. What kind of geometric object is formed by the intersection of this sphere with the `yz`-plane? Find the equations that represent this intersection.
  6. Find the equation of a sphere that goes through `(1,-3,-4)` with center `(-7,-5,2)`. Use your work from number 4.
    1. Sketch one period of the curve `y = sin x` in the `xy`-plane.
    2. Sketch one period of the cylinder `y = sin x` in `RR^3`.
    3. Sketch one period of the cylinder `z = sin x` in `RR^3`.
  8. Sketch the cylinders:
    1. `z = e^y`
    2. `x^2/4 + y^2/9 = 1`
